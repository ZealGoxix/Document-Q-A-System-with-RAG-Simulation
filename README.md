# Document-Q-A-System-with-RAG-Simulation
Document Q&amp;A System with RAG Simulation

How to Run and Learn
Set up the project:

bash
mvn clean install
mvn spring-boot:run
Access the application:

Main page: http://localhost:8080

Upload: http://localhost:8080/documents

Q&A: http://localhost:8080/qa

H2 Console: http://localhost:8080/h2-console
